## 1. المشروع باختصار

نظام **GFRC** (Government Financial Receipt & Cash Ledger) هو نظام حكومي متكامل لإدارة الإيصالات المالية والمقبوضات النقدية. يتيح النظام إنشاء سجلات مالية ديناميكية (Dynamic Registers)، إصدار وصولات متسلسلة، تتبع المراجعات، وإنشاء تقارير مالية شاملة مع سجل تدقيق كامل.

### الروابط
| الخدمة | الرابط |
|--------|--------|
| الواجهة الأمامية | `http://localhost:5173` |
| API الباك إند | `http://localhost:8000/api/v1` |
| تسجيل الدخول | `admin` / `Admin@12345` |

## 4. دورة حياة الوصل (Receipt Lifecycle)

```
[إنشاء] ──▶ DRAFT (مسودة)
              │
              ▼
         PENDING (معلقة)
              │
    ┌─────────┴─────────┐
    ▼                   ▼
 ISSUED (مرحل)     CANCELLED (ملغى)
    │                   ↑
    ▼                   │
 PRINTED (مطبوع)   [يتطلب سبب + صلاحية]
    │
    ▼
[QR Code + PDF]
```

| الحالة | الوصف | الإجراءات المتاحة |
|--------|-------|-------------------|
| **DRAFT** | مسودة جديدة | تعديل، ترحيل، إلغاء |
| **PENDING** | قيد الانتظار | تعديل، ترحيل، إلغاء |
| **ISSUED** | مرحل نهائياً | طباعة، تعديل (بمراجعة)، إلغاء |
| **PRINTED** | تمت الطباعة | تعديل (بمراجعة)، إلغاء |
| **CANCELLED** | ملغى | عرض فقط (لا تعديل) |

## 8. مثال عملي: إنشاء وصل غرامة مرورية

### الخطوة 1: اختيار السجل
- اذهب إلى **الوصولات** ← **وصل جديد**
- اختر **سجل الغرامات** من القائمة

### الخطوة 2: ملء الحقول
| الحقل | القيمة |
|-------|--------|
| رقم المخالفة | `V-2026-001` |
| تاريخ المخالفة | `2026-05-27` |
| نوع المخالفة | تجاوز السرعة المحددة |
| رقم لوحة المركبة | `بغداد 12345` |
| مبلغ الغرامة | `25000` |
| مبلغ الخصم | `0` |
| اسم الضابط | `أحمد محمد` |
| ملاحظات | `تم الدفع نقداً` |

### الخطوة 3: الحفظ
- اضغط **حفظ**
- سيُنشأ الوصل برقم: `FINE-2026-000001`
- المجموع الكلي: `25000.000` دينار

### الخطوة 4: الترحيل
- في صفحة تفاصيل الوصل، اضغط **ترحيل**
- سيُولد QR Code تلقائياً
- الوصل يصبح غير قابل للتعديل (بدون مراجعة)

### الخطوة 5: الطباعة
- اضغط **طباعة**
- تظهر صفحة A4 تحتوي:
  - اسم الدائرة (من الإعدادات)
  - رقم الوصل الكبير
  - QR Code (أعلى اليمين)
  - تفاصيل الحقول
  - المجموع بالأرقام والكتابة
  - كود التحقق + ختم النظام

## 11. التعليمات السريعة

### إضافة سجل جديد
```
السجلات ← + سجل جديد ← أدخل الكود والاسم والسنة ← حفظ
```

### إضافة حقل لسجل
```
السجلات ← اختر السجل ← إضافة حقل ← املأ البيانات ← حفظ
```

### إنشاء وصل
```
الوصولات ← وصل جديد ← اختر السجل ← املأ الحقول ← حفظ ← ترحيل ← طباعة
```

### تصدير تقرير
```
التقارير ← اختر النوع ← حدد التواريخ ← تصدير CSV
```

### نسخ احتياطي
```
الإعدادات ← النسخ الاحتياطي ← إنشاء نسخة ← تحميل
```
